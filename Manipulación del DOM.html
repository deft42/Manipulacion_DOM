Manipulación del DOM (Document Object Model).

La Manipulación del DOM (Document Object Model) es un concepto fundamental en desarrollo web que se refiere a la forma en que puedes interactuar y cambiar elementos en una página web utilizando JavaScript. Para explicarlo de manera sencilla, imagina que una página web es como una casa y el DOM es como un plano detallado de esa casa. Puedes usar JavaScript para agregar, quitar o modificar objetos dentro de esa casa (la página web) según tus necesidades.

Pero antes debemos entender algunos conceptos fundamentales y que son parte de la sintaxis basica de Javascript esto poder accesar a los elementos que nosotros queremos modificar sin afectar de forma negativa el resto.

Pimero que nada si bien ya hicimos mención que el DOM es un objeto en un documento . Esto quiere decir que en dicho objeto hay elementos padres, primos e hijos (Parent, Siblings & Children). Dicho de otra manera, imagina que estás construyendo un árbol, pero en lugar de ramas y hojas, tienes elementos HTML y JavaScript. En este caso, los "elementos padres" son como ramas principales, y los "elementos hijos" son como las ramitas que cuelgan de esas ramas.

Visto de forma grafica, seria asi:

Esta es la manera en como Javascript representa el documento HTML en nuestro navegador y la forma en la que representa los elementos que la contienen.

window es como la ventana principal de tu navegador. Piensa en ello como el entorno global donde vive tu código JavaScript cuando se ejecuta en un navegador web.

document se refiere al objeto global que representa el documento HTML cargado en la ventana. En términos más sencillos, es como el lienzo en el que está pintada toda la página web.

HTML. Es como el mapa o la representación estructurada de tu página HTML. Piensa en HTML como la estructura y contenido de tu página web.

Elemento Padre: Este es como el "abuelo" de todos. Un elemento padre contiene otros elementos. Piensa en un <div> como un padre que tiene hijos, como un <p> y un <img>. El <div> es el jefe de familia aquí.

Elemento Hijo: Estos son como los hijos en una familia. Viven dentro de un elemento padre. Así que, si tienes un <div> que contiene un <p>, el <p> es el hijo del <div>. Puedes tener varios hijos, como un montón de <p> o cualquier otro elemento.

Elemento Primo: Ahora, los primos son un poco más lejanos en la familia, pero aún están conectados. En términos de HTML y JavaScript, dos elementos son primos si comparten el mismo abuelo. Si tienes un <div> que tiene un <p> como hijo, y otro <div> que también tiene un <p> como hijo, esos dos <p> son primos. Comparten el mismo abuelo <div>.

Ejemple:

<div> <!-- Este es el elemento padre y tengo un solo hijo -->
    <p>¡Hola! Soy el hijo.</p> <!-- Este es el elemento hijo -->
</div>

<div> <!-- Otro elemento padre Y tengo tres hijos (primo del anterior) -->
		<img> ¡Hola de nuevo! Soy otro hijo.</p> <!-- Otro elemento hijo (primo del anterior) -->
		<h1></h1> ¡Hola Soy otro hijo.</p> <!-- Otro elemento hijo (primo del anterior) -->
    <p>Soy otro hijo.</p> <!-- Otro elemento hijo (primo del anterior) -->
</div>

Selectores en Javascript.

Los selectores en JavaScript son herramientas que te permiten seleccionar y manipular elementos HTML desde tu código. Son como una especie de atajos que te ayudan a identificar y trabajar con partes específicas de tu página web.

Hay varios tipos de selectores, pero los más comunes son:

Selectores por etiqueta: Seleccionan todos los elementos con una etiqueta específica.

Sintaxis:

// Selecciona todos los elementos con etiqueta p dentro del HTML
document.getElementsByTagName('p');
// Seleccionara todos los parrafos que contenga la pagina.

Selectores por clase:: Seleccionan elementos que tienen una clase específica.

Sintaxis:

// Selecciona todos los elementos que tengan la clase mi-clase en su etiqueta.
document.getElementsByClassName('mi-clase');

Selectores por ID: Seleccionan un elemento con un ID específico.

// Seleccionara el elemento que en su etiqueta tenga el ID mi-id
document.getElementById('mi-id');

Hay que recordar que las etiquetas que contengan un ID dentro de ellas son unicos, es decir, no se pueden repetir en cuanto a nombre.

Selectores por atributo: Seleccionan elementos que tienen un atributo específico.

// Seleccionará todos los elementos con el atributo y valor especificados.
document.querySelectorAll('[atributo=valor]');

Selectores por relación: Seleccionan elementos basados en su relación con otros elementos.

// eleccionará el primer <span> que sea hijo directo de un <p>.
document.querySelector('p > span');

Eventos en Javascript.

Los eventos en JavaScript son como señales que tu página web puede detectar y responder a medida que interactúas con ella. Imagina que estás haciendo clic en un botón, moviendo el ratón, o escribiendo en un campo de texto. Cada una de estas acciones es un evento, y JavaScript te permite escribir código que reaccione a estos eventos.

Aquí hay algunos conceptos clave:

Eventos Básicos:

  • Clic (Click): Ocurre cuando haces clic en un elemento.
  • Doble Clic (DoubleClick): Ocurre cuando haces doble clic en un elemento.
  • Tecla Presionada (KeyPress): Ocurre cuando presionas una tecla del teclado.
  • Cambio (Change): Ocurre cuando el valor de un elemento cambia, como en un campo de formulario.
  • Scroll: Se activa cuando el usuario desplaza la página hacia arriba o hacia abajo. Perfecto para implementar efectos de desplazamiento.
  • Mouseover y Mouseout: Estos eventos se desencadenan cuando el cursor del mouse entra o sale de un elemento. Útiles para efectos visuales o información adicional.
  • Submit: Ocurre cuando se envía un formulario. Puedes usarlo para validar datos antes de enviarlos al servidor.
  • Focus y Blur: El evento de enfoque ocurre cuando un elemento obtiene el foco (por ejemplo, un campo de entrada), y el evento de desenfoque cuando pierde el foco. Útiles para validaciones en formularios.
  • DOMContentLoaded: Este evento se activa cuando el documento HTML ha sido completamente cargado y analizado, sin esperar a que se carguen imágenes y hojas de estilo. Es útil para realizar acciones iniciales cuando la estructura del documento está lista.

Escuchadores de Eventos: Para manejar un evento, usas "escuchadores" (listeners) de eventos.

Es un método en JavaScript que se utiliza para adjuntar un evento a un elemento del DOM (Documento Object Model). Este método permite escuchar eventos y ejecutar una función específica cuando ocurre ese evento.

Sintaxis basica: 

elemento.addEventListener(evento, función);
// El evento puede ser cualquiera de los que se hicieron mencion arriba, es decir,
// puede ser un evento clic, keyPress, Scroll, etc.
// La función representa la instruccion al accionar el dicho evento, es decir, 
// que queremos que haga cuando el usuario cliquee, presione una telca o haga scroll.
// Creamos una variable llamada miBoton. 
// Seleccionamos al elemento dentro del documento por su nombde de ID.
var miBoton = document.getElementById('miBoton');

// Creamos una instruccion que escuche y realize un evento al hacer "clic"
miBoton.addEventListener('click', function() {
    // Una ves que se ha hecho clic, se va a activar una funcion sin parametros
		// la cual tendra una instruccion de desplegar un mensaje dentro del navegador.
    alert('¡Se hizo clic en el botón!');
});

Ejemplos practicos.

Cambiar de color el fondo con javascript.

HTML

<!-- Botón para cambiar el fondo -->
  <button id="cambiarFondoBtn">Cambiar Fondo</button>

CSS

body {
      display: flex;
      align-items: center;
      justify-content: center;
      height: 100vh;
      margin: 0;
      transition: background-color 0.5s ease;
  }

Javascript

// Creamos una variable y Seleccionamos el botón por su ID
    let boton = document.getElementById('cambiarFondoBtn');

    // Agregamos un evento de clic al botón y al mismo tiempo agregamos
    // una funcion sin parametro entre los parentesis, pero si con una instruccion dentro
    // de ella entre los corchetes.
    boton.addEventListener('click', function() {
      // Verificamos el color actual del fondo entrando a la ventana del navegador y tomando
      // las propiedades que tenga, es decir el CSS.
      // Para esto, vamos a crear una variable que va a contener todos estos datos, la cual 
      // llamaremos colorActual
      let colorActual = window.getComputedStyle(document.body, null).getPropertyValue('background-color');

      // Si el color actual es, en este caso, identico a blanco, cambiamos a azul; de lo contrario, volvemos a blanco.
      // rgb es un modo de color que pueden usar en CSS al igual que valores hexadecimales.
      // RGB = Red, Blue, Green. En este caso 255, 255, 255. Corresponde al color blanco puro.
      // RGBA = Red, Blue, Green, Aplha. Esta propiedad es similar a la anterior, la diferentcia es que
      // el ulstimo valor hace referencia a la transparencia del color. La cual podemos media con 
      // valores numericos decimales y enteros, es decir, 0 representaria la ausencia de color, 1 la totalidad del color
      // Y los numeros decimales, el % de transparencia, por ejemplo, .5 = 50% de transparencia.
      
      if (colorActual === 'rgb(255, 255, 255)') {
        document.body.style.backgroundColor = 'blue';
      } else {
        document.body.style.backgroundColor = 'white';
      }
    });

Suma de dos numeros ingresados por el usuario en campo de texto.

<form id="additionForm">
    <label for="num1">Primer numero:</label>
    <input type="text" id="num1">
    <br>

    <label for="num2">Segundo numero:</label>
    <input type="text" id="num2">
    <br>

    <button type="button" onclick="performAddition()">Sumar</button>
  </form>

  <p id="result"></p>

Javascript

// Creamos una funcion que se llama "realiza Suma"
function realizaSuma() {
      // Obtenemos los valores que ingresemos en los campos de texto
      // Para ello creamos dos contantes, una para cada digito.
      // En este caso son constantes por que siempre queremos que sea
      // un digito en los campos.
  
      // Accesamos a los elementos buscandolos en el documento por su ID y
      // los guardamos en sus respectivas constantes.
      const num1 = parseFloat(document.getElementById('num1').value);
      // parseFloat, Se utiliza para aceptar una cadena y convertirla en un número
      // de punto flotante, de esta manera aseguramos que todos los datos
      // sean de tipo numero.
      const num2 = parseFloat(document.getElementById('num2').value);

      // Verificamos los campos y aseguramos que solo sean numeros los que
      // se ingresen.
      
      if (!isNaN(num1) && !isNaN(num2)) {
        // Si la condicion se cumple, pasamos a realiza la suma
        const resultado = num1 + num2;

        // Desplegamos el resultado en pantalla
        document.getElementById('result').innerText = `El resultado es: ${resultado}`;
      } else {
        // Desplegamos un mensaje de error, en el caso de que no sea un numero
        document.getElementById('result').innerText = 'Por favor, solo ingrese numeros.';
      }
    }

Cambiar de color y contenido un boton

HTML

<button id="enviarBtn">Enviar</button>

CSS

body {
  height: 100%;
  width: 100%;
  display: flex;
  justify-content: center;
  align-self: center;
}

.enviado {
      background-color: rgba(46, 204, 113,1.0);
    font-weight: bold;
  color: #ffffff;
  font-size; 16px;
  border:none;
    }

button {
  background-color: rgba(52, 152, 219,1.0);
  padding: 15px 35px;
  font-weight: bold;
  color: #ffffff;
  font-size; 16px;
  border:none;
  
}

Javascript

// Seleccionamos el botón por su ID
    var enviarBtn = document.getElementById('enviarBtn');

    // Agregamos un evento de clic al botón
    enviarBtn.addEventListener('click', function() {
      // Cambiamos el texto del botón a 'Enviado'
      enviarBtn.innerText = 'Enviado';

      // Aplicamos la clase 'enviado' para cambiar el color
      enviarBtn.classList.add('enviado');

      // Deshabilitamos el botón después de hacer clic
      enviarBtn.disabled = true;
    });
